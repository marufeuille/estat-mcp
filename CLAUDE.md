# CLAUDE.md — estat-mcp プロジェクト行動指針

## 0. プロジェクト管理
- Projectに関するドキュメントはObsidianのDev vaultで管理されている。
- ObsidianへはObsidian CLIでのアクセスできる


- storyの一覧

```
Obsidian vault=Dev files folder=Projects/estat-mcp/stories/
```

- storyにアクセスしたい場合

```
Obsidian read path=Projects/estat-mcp/stories/{story-slug}.md
```

- storyの中のタスクにアクセスしたい場合

```
Obsidian read path=Projects/estat-mcp/tasks/{story-slug}/{task-slug}.md
```

- task完了時はstoryに記載のあるタスクのチェックボックスにチェックを入れること
